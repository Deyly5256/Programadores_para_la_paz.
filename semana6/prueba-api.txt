 Prueba de la ruta POST /registro
=================================

Se envió una petición POST a: http://localhost:3000/registro

Con el siguiente JSON en el cuerpo (body):
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor devolvió la siguiente respuesta:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Explicación:
El servidor recibe los datos enviados por el cliente en formato JSON.
Gracias al middleware express.json(), Express puede leer el cuerpo de la
petición POST y almacenarlo en req.body. Luego extrae req.body.nombre y
req.body.mensaje, y los devuelve como confirmación de que los datos
fueron recibidos correctamente.
